평택 베리타스학원
01
평서문
Children today / are / "safe"
오늘날 아이들은 “안전하다”.
02
구성 동사 + 목적어 묶음
Their only worries / consist of / fights with their parents / going to school / and dealing with the list of activities / that they're signed up for
그들의 유일한 걱정은 … 처리하는 일이다.
03
의문문
But / what's / going on / ?
그런데 무슨 일이 벌어지고 있는 걸까?
04
조건절 + 의문문
If / they're / "safe" / and living / "well," / why / do they experience / so many problems / ?
그들이 “안전”하고 “잘” 산다면 왜 많은 문제를 겪을까?
05
현재완료 + 관계사절
Children / have learned / to be cautious; / a lot of kids / have / even lost / that innate curiosity / that no one should ever lose
아이들은 조심하는 법을 배웠고; 많은 아이들은 타고난 호기심을 잃었다.
